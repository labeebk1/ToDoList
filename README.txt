To Do List Web Application

Author: Labeeb Khan
Date: July 24, 2018

Description:

The following is a Web Application for a simple To-Do List. 
This application supports the following functionalities: Add, Modify, and Delete Entries in a To-Do List

Program Instructions:

To run the program, please run the following command:

python manage.py runserver

Program Assumptions:

You must have Django 1.11 and Python 3.6.4 Installed
